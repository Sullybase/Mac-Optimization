# Mac Optimization

My Mac apps I use for productivity and more

## What

This is a curated selection of Mac apps. All are free, and most are open source on GitHub. I use these on macOS 27.

## General

- [Aerial](https://github.com/JohnCoates/Aerial) — Custom animated wallpaper and screen saver
- [BoringNotch](https://github.com/TheBoredTeam/boring.notch) — Adds media controls, volume/battery notifications, calendar, and file tray to the notch
- [Disk Space Analyzer](https://nektony.com/disk-expert) — Free disk analyzer
- [Everyday Habit Tracker](https://everyday.app) — Simple daily to-dos (mine is Code, Read, Puzzle, for example)
- [Grammarly](https://www.grammarly.com/native/mac) — Why not
- [IconChanger](https://github.com/Bengerthelorf/macIconChanger) — Custom app icons

## Privacy

- [Ghostery](https://www.ghostery.com/) — Safari ad/tracker block extension
- [Proton Pass, Mail & VPN](https://proton.me/) — Free, Swiss-backed, privacy-first VPN and encrypted mail/files
- [uBlock Origin Lite](https://apps.apple.com/app/ublock-origin-lite/id6745342698) — Safari ad/tracker block extension
- [Vivaldi](https://vivaldi.com/) — European browser, privacy and customizable

## Productivity

- [Latest](https://github.com/mangerlahn/Latest) — Non-App Store app updater
- [LaunchOS](https://launchosapp.com) — The removed app launchpad that Spotlight replaced
- [Launchy](https://apps.apple.com/us/app/launchy-app-launcher/id6739782043) — App/folder launch wheel at mouse
- [Nightshift](https://yep.github.io/Nightshift/) — Safari dark mode extension
- [Numi](https://numi.app/) — Simple calculator
- [PDFgear: PDF Editor & Reader](https://www.pdfgear.com/) — PDF editor, iOS too
- [Pixen](https://pixenapp.com/) — Pixel art editor
- [Raycast](https://www.raycast.com/) — Spotlight replacement
- [SuperWhisper](https://superwhisper.com/) — Dictation replacement with better local AI and faster

## Utility

- [GIMP](https://www.gimp.org/) — Any format free image editor
- [Karabiner-Elements](https://karabiner-elements.pqrs.org/) — External keyboard mapping
- [Mac Mouse Fix](https://macmousefix.com/) — External mouse mapping
- [LocalSend](https://localsend.org/) — Any device file share
- [MonitorControl](https://github.com/MonitorControl/MonitorControl) — External monitor controls for brightness and more
- [OPAutoClicker](https://opautoclicker.com/) — Auto clicker
- [Pearcleaner](https://github.com/alienator88/Pearcleaner) — App remover that gets all left-behind files
- [ProNotes](https://www.pronotes.app) — Apple Notes upgrade for formatting and more
- [Stats](https://github.com/exelban/stats) — RAM, GPU, CPU, heat, and more monitor
- [SuperCorners](https://github.com/daniyalmaster693/SuperCorners) — Better Mac hot corners
- [Thaw](https://github.com/stonerl/Thaw) — Menu bar item hider
- [Upscayl](https://upscayl.org/) — Local AI image upscaler
- [VLC](https://www.videolan.org/vlc/) — Media player
- [Battery](https://github.com/actuallymentor/battery) - Battery limiter
- [KeepingYouAwake](https://github.com/newmarcel/KeepingYouAwake) - Prevents display from turning off
